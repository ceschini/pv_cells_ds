main functions:

MYRPCA.m(executable): the main driven function of the evaluation framework
MYCROP.m: preprocessing code to crop the solar panels out from their soil background
MYTRANSFORM.m: wrapper function for the TILT algorithm
RobustPCA.m: the main function to perform Robust PCA
TILT.m: the main function to perform TILT

data:

in total 120 PV images in the target folder
